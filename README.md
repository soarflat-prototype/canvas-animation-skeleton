# Canvas Animation Skeleton
Canvas Animation Skeleton is a skeleton for canvas animation.

## Install
```
yarn install
```

## How to use scripts

### server
```
yarn run server
```

### build
```
yarn run build
```

#### build:css
```
yarn run build:css
```

#### build:js
```
yarn run build:js
```

### watch
```
yarn run watch
```

#### watch:css
```
yarn run watch:css
```

#### watch:js
```
yarn run watch:js
```
